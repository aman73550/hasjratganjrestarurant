# Hazratgunj Restaurant & Banquet Hall — Landing Page

## Overview
A stunning luxury landing page for Hazratgunj Restaurant and Banquet Hall, Gorakhpur's first wall-lit boutique banquet hall. Built with React + Vite + TypeScript + Tailwind CSS + Framer Motion.

## Design System
- **Color Palette:** Deep dark backgrounds (#060402, #0A0804), Royal Gold (#D4AF37), off-white/ivory text
- **Typography:** Playfair Display (serif headlines), Montserrat (body text)
- **Theme:** Midnight & Champagne — ultra-luxurious dark mode aesthetic
- **Glassmorphism:** Navigation bar + card components use backdrop blur glass effect

## Sections
1. **Hero** — Full-screen parallax with banquet hall image, headline, CTAs, and stats
2. **Banquet Experience** (#banquet) — 600+ capacity, underground parking, wall-lit interiors
3. **Culinary Excellence** (#cuisine) — Signature Awadhi dishes + food photography
4. **Luxury Amenities** (#amenities) — 6 amenity cards with icons
5. **Venue Gallery** (#gallery) — 6 venue/hotel images in a grid with hover zoom + lightbox
6. **Food Gallery** (#gallery) — 6 food images in a grid with hover zoom + lightbox
7. **Social Proof** (#reviews) — 4.9/5 rating, 4 rotating review cards
8. **Contact & Location** (#contact) — Phone, WhatsApp, address, embedded Google Maps
9. **Footer** — Quick links, social icons, developer credit (Aman Gupta)

## Mobile Features
- Glassmorphism mobile menu (full-screen overlay)
- Sticky bottom action bar: Call / WhatsApp / Directions / Book Now
